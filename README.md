<h1>Requisitos</h1>
Al descargar el codigo tienes que tener esto instalado:
-nodejs

-npm

<h1>Instrucciones</h1>
Al descargar el codigo, dentro de la carpeta To Do App tienes que instalar electron que es con el siguiente comando:
-npm install electron

Y nada mas
